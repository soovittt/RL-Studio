# Landing Page Images

Place the following images in this directory:

1. `environment-builder.png` - Main RL Studio interface showing the environment editor with 3D grid canvas, object palette, and controls
2. `rollout-visualizer.png` - Rollout preview showing 3D grid environment visualization with rollout controls and metrics
3. `training-config.png` - New Training Run modal showing training configuration options (PPO, hyperparameters, GPU settings)
4. `export-dialog.png` - File save dialog showing export functionality

All images should be in PNG format and optimized for web use.
